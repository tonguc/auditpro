# AuditPro — Website Audit Tool

Thank you for purchasing AuditPro! This is a fully offline, interactive website audit tool covering SEO, UX, Conversion, and AI visibility.

## How to Open

1. Extract the ZIP file
2. Double-click **index.html**
3. Open in **Chrome** or **Edge** (recommended)

> **Tip:** If the page appears blank, right-click `index.html` → Open with → Google Chrome.

## What's Included (Free / Lite)

- **18 hand-picked audit checks** across 5 categories:
  - Technical SEO (HTTPS, page speed, schema, broken links)
  - On-Page & Content (title, meta, H1, keyword)
  - UX Heuristics (navigation, visual hierarchy, user language, task completion)
  - Conversion & CTA (CTA visibility, trust signals, form friction)
  - AI & SERP Visibility (featured snippets, FAQ schema, content structure)
- Pass / Partial / Fail / N/A scoring for each check
- Weighted score dashboard (0–100) with grade (A–D)
- Top priority issues list
- **PDF export** — download a branded audit report
- Results saved locally in your browser
